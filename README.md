# Sultan Of Sweetness

A luxury dessert brand website built with React, showcasing premium brownies, cookies, and event catering in a polished, elegant style.

## Live Demo
https://sultanofsweetness.netlify.app/

## Features
- Responsive landing page
- Elegant luxury dessert branding
- Mobile navigation menu
- Section-based layout for menu, about, gallery, and contact
- Deployed on Netlify

## Tech Stack
- React
- CSS3
- JavaScript
- React Icons
- Netlify
